# 管理者後台 報表 Dashboard

提供集團飯店的報表數據等。

## Setup

安裝套件工具包

```bash
# bun
bun install
```

## Development Server

啟動伺服器在 `http://localhost:3000`:

```bash
# bun
bun run dev
```

## Production

建構應用程式:

```bash
# bun
bun run build
```

本地端預覽生產版本:

```bash
# bun
bun run preview
```


## Node.js version

```bash
node 版本 v18.0.0 +
```


## 啟動開發指令

此腳本假設您的系統已安裝 `nvm` 和 `bun`

```bash
./start_dev.sh
```
